# AI Theme Visual Reference Guide\n\n## Design System Overview\n\nThis guide documents the complete AI theme styling system used across the FleetNeuron application, with before/after comparisons for the three redesigned pages.\n\n---\n\n## Core Design Elements\n\n### 1. Glass Morphism Panels\n\n**Purpose:** Main content containers with modern frosted glass effect\n\n**Styling:**\n```css\n.ai-panel {\n  border-radius: 16px;\n  padding: 14px;\n  background:\n    radial-gradient(circle at 0% 0%, rgba(45, 212, 191, 0.11), transparent 45%),\n    radial-gradient(circle at 100% 100%, rgba(59, 130, 246, 0.16), transparent 50%),\n    rgba(15, 23, 42, 0.95);\n  border: 1px solid rgba(51, 65, 85, 0.9);\n  box-shadow:\n    0 16px 35px rgba(2, 6, 23, 0.65),\n    0 0 0 1px rgba(15, 23, 42, 0.9);\n  backdrop-filter: blur(14px);\n}\n```\n\n**Visual Characteristics:**\n- Dark navy background (#0f172a) with 95% opacity\n- Two radial gradients: Teal (top-left), Blue (bottom-right)\n- 14px blur backdrop filter for glass effect\n- Double box-shadow: Outer (16px) and inner (1px) for depth\n- 16px border radius for modern appearance\n\n**Used In:** All page sections, stat cards, filter containers\n\n---\n\n### 2. Header Section\n\n**Purpose:** Page title, description, and primary action area\n\n**Components:**\n```html\n<header class=\"ai-header\">\n  <div>\n    <p class=\"ai-kicker\">FleetNeuron AI [Section]</p>\n    <h1>\n      <span class=\"material-symbols-outlined title-icon\">[icon]</span>\n      Page Title\n    </h1>\n    <p class=\"subtitle\">Description text</p>\n  </div>\n  <!-- Primary CTA button -->\n</header>\n```\n\n**Styling Breakdown:**\n\n| Element | Color | Font Size | Properties |\n|---------|-------|-----------|------------|\n| **Kicker** | #67e8f9 (cyan) | 0.72rem | UPPERCASE, 0.14em letter-spacing |\n| **Title** | #f8fafc (white) | clamp(1.45rem, 2.1vw, 2rem) | Flex with icon, responsive |\n| **Icon** | #38bdf8 (sky) | 1em | Colored accent |\n| **Subtitle** | #94a3b8 (slate) | inherit | Muted secondary text |\n\n**Flexbox Layout:** Space-between, flex-start alignment, 16px gap\n\n**Used In:** Every redesigned page header\n\n---\n\n### 3. Glowing Action Button\n\n**Purpose:** Primary call-to-action buttons with emphasis\n\n**Styling:**\n```css\n.btn-primary {\n  background: linear-gradient(135deg, #2563eb 0%, #1d4ed8 50%, #0ea5e9 100%);\n  color: #fff;\n  border: none;\n  border-radius: 10px;\n  padding: 8px 12px;\n  font-weight: 600;\n  cursor: pointer;\n  transition: transform 120ms ease, filter 120ms ease;\n}\n\n.btn-primary:hover {\n  transform: translateY(-1px);\n}\n\n.ai-glow-btn {\n  box-shadow:\n    0 10px 24px rgba(37, 99, 235, 0.45),\n    inset 0 0 0 1px rgba(125, 211, 252, 0.28);\n}\n```\n\n**Visual Effects:**\n- **Gradient:** Blue (#2563eb) → Deep Blue (#1d4ed8) → Sky (#0ea5e9)\n- **Outer Glow:** 10px blue shadow with 45% opacity\n- **Inset Border:** Cyan (#7dd3fc) highlight inside button\n- **Hover:** Subtle upward movement (-1px)\n- **Icon Gap:** 7px spacing between icon and text\n\n**Button Sizes:**\n- Standard: 8px vertical, 12px horizontal padding\n- Small (.btn-sm): 5px vertical, 9px horizontal padding\n- Medium (.btn-secondary): Same padding, gray styling\n\n**Used In:** \"Sync ELD\", \"New Invoice\", \"Export Data\", \"Save\" actions\n\n---\n\n### 4. Status Badges & Pills\n\n**Purpose:** Show status at a glance with color-coded indicators\n\n**Structure:**\n```html\n<span class=\"status-pill\">\n  <span class=\"dot\"></span>\n  Status Text\n</span>\n```\n\n**Styling:**\n```css\n.status-pill {\n  display: inline-flex;\n  align-items: center;\n  gap: 6px;\n  padding: 3px 8px;\n  border-radius: 999px;\n  border: 1px solid rgba(34, 197, 94, 0.7);\n  color: #86efac;\n  text-transform: uppercase;\n  font-size: 0.7rem;\n  letter-spacing: 0.08em;\n}\n\n.status-pill .dot {\n  width: 7px;\n  height: 7px;\n  border-radius: 999px;\n  background: #22c55e;\n  box-shadow: 0 0 8px rgba(34, 197, 94, 0.9);\n}\n```\n\n**Color Variants:**\n\n| Status | Border Color | Text Color | Dot Color | Shadow |\n|--------|--------------|-----------|-----------|--------|\n| Active/Compliant | Green (34, 197, 94) | #86efac | #22c55e | 8px green glow |\n| Inactive/Violation | Red (248, 113, 113) | #fca5a5 | #ef4444 | 8px red glow |\n| Pending | Amber (217, 119, 6) | #fed7aa | #d97706 | 8px amber glow |\n| Draft | Slate (100, 116, 139) | #cbd5e1 | #64748b | 8px slate glow |\n\n**Styling Tips:**\n- Animated dot with matching color glow\n- UPPERCASE text with tight letter-spacing\n- Pill-shaped (999px border radius)\n- Very small font (0.7rem) for compactness\n\n**Used In:** HOS status, Vehicle compliance status, Invoice payment status\n\n---\n\n### 5. Form Inputs & Selects\n\n**Purpose:** Consistent input styling with focus states\n\n**Default State:**\n```css\n.form-input, .form-select {\n  width: 100%;\n  border: 1px solid rgba(71, 85, 105, 0.95);\n  border-radius: 10px;\n  padding: 10px 12px;\n  color: #e5e7eb;\n  background:\n    radial-gradient(circle at 0 0, rgba(59, 130, 246, 0.08), transparent 52%),\n    rgba(2, 6, 23, 0.8);\n  outline: none;\n  transition: border-color 150ms ease, box-shadow 150ms ease;\n}\n```\n\n**Focus State:**\n```css\n.form-input:focus, .form-select:focus {\n  border-color: rgba(56, 189, 248, 0.95);\n  box-shadow: 0 0 0 3px rgba(14, 165, 233, 0.2);\n}\n```\n\n**Visual Characteristics:**\n- Light blue radial gradient on dark background\n- 10px border radius for modern look\n- Blue focus state with outer ring (3px)\n- Smooth 150ms transitions\n- 10px padding for touch-friendly sizing\n\n**Used In:** Invoice filters, HOS search, Audit category selector\n\n---\n\n### 6. Data Tables\n\n**Purpose:** Display structured data with consistent styling\n\n**Header Row:**\n```css\n.table th {\n  background: rgba(2, 6, 23, 0.85);\n  color: #a5b4fc;\n  font-size: 0.75rem;\n  text-transform: uppercase;\n  letter-spacing: 0.08em;\n  padding: 10px;\n  border-bottom: 1px solid rgba(51, 65, 85, 0.45);\n}\n```\n\n**Body Rows:**\n```css\n.table td {\n  color: #e2e8f0;\n  font-size: 0.9rem;\n  padding: 10px;\n  border-bottom: 1px solid rgba(51, 65, 85, 0.45);\n  white-space: nowrap;\n}\n```\n\n**Table Wrapper:**\n```css\n.table-wrap {\n  overflow: auto;\n  border-radius: 12px;\n  border: 1px solid rgba(51, 65, 85, 0.8);\n}\n```\n\n**Visual Hierarchy:**\n- Very dark header (#0f172a with 85% opacity)\n- Indigo text for column names (#a5b4fc)\n- Slate text for data (#e2e8f0)\n- Subtle row dividers (1px borders)\n- 10px padding for readability\n- Horizontal scrolling on mobile\n\n**Used In:** HOS records, Invoices list, Audit trail\n\n---\n\n## (#2563eb):** Primary buttons, selected states\n\n### Status Colors\n- **Green (#22c55e):** Success, active, compliant, in-service\n- **Amber (#d97706):** Warning, pending, caution states\n- **Red (#ef4444):** Error, violation, danger, overdue\n- **Slate (#64748b):** Neutral, draft, inactive states\n\n### Neutral Colors\n- **Navy (#0f172a):** Main background\n- **Slate-900 (#0f172a - 0.95):** Panel backgrounds\n- **Slate-700 (#475569):** Borders, secondary elements\n- **Slate-400 (#94a3b8):** Muted text\n- **Slate-100 (#f1f5f9):** Light text\n\n### Opacity Patterns\n- **0.95** - Panel backgrounds (slight transparency)\n- **0.85** - Table headers\n- **0.8** - Input backgrounds\n- **0.7** - Borders and dividers\n- **0.45** - Box shadows\n- **0.28** - Inset button borders\n- **0.2** - Focus ring backgrounds\n- **0.12** - Stat card gradients\n- **0.11** - Panel gradients (teal)\n- **0.16** - Panel gradients (blue)\n\n---\n\n## Responsive Design Breakpoints\n\n**Mobile First Approach:**\n\n```css\n/* Default: Mobile (< 720px) */\n.filters-grid { grid-template-columns: 1fr; }\n.stats-grid-4 { grid-template-columns: repeat(2, 1fr); }\n\n/* Tablet (720px - 1120px) */\n@media (min-width: 720px) {\n  .requirements-grid { grid-template-columns: repeat(2, 1fr); }\n}\n\n/* Desktop (1120px+) */\n@media (min-width: 1120px) {\n  .ai-header { flex-direction: row; }\n  .stats-grid-4 { grid-template-columns: repeat(4, 1fr); }\n}\n```\n\n**Key Breakpoints:**\n- **320px:** Extra small phones\n- **768px:** Tablets\n- **1024px:** Large tablets / small laptops\n- **1440px:** Desktop\n- **1920px:** Ultra-wide displays\n\n---\n\n## Typography System\n\n**Font Scale (Relative to 16px base):**\n\n| Size | Value | Usage |\n|------|-------|-------|\n| Kicker | 0.72rem | Section labels |\n| Small | 0.75rem | Table headers, badge labels |\n| Caption | 0.8rem | Secondary text, violations |\n| Small Text | 0.85rem | Form labels |\n| Body | 0.9rem | Table cells, action text |\n| Subtitle | inherit | Muted secondary text |\n| Title | clamp(1.45rem, 2.1vw, 2rem) | Responsive page titles |\n| Value | 1.8rem | Stat values |\n\n**Font Weights:**\n- **400:** Regular body text\n- **600:** Buttons, action items\n- **700:** Stat values, important numbers\n\n**Letter Spacing:**\n- Standard: 0.01em (subtle)\n- Kicker: 0.14em (wide, uppercase effect)\n- Headers: 0.08-0.09em (elegant spacing)\n- Labels: 0.04em (tight, compact)\n\n---\n\n## Animation & Transitions\n\n**Button Interactions:**\n```css\nbutton {\n  transition: transform 120ms ease, filter 120ms ease;\n}\n\nbutton:hover {\n  transform: translateY(-1px); /* Subtle lift */\n}\n```\n\n**Focus States:**\n```css\ninput:focus {\n  transition: border-color 150ms ease, box-shadow 150ms ease;\n}\n```\n\n**Animations:**\n- All transitions: 120-150ms\n- Easing: `ease` (smooth acceleration/deceleration)\n- Movement: Subtle (-1px vertical)\n- Shadow: Smooth color transition\n\n---\n\n## Accessibility Considerations\n\n✅ **High Contrast:** All text meets WCAG AA (4.5:1 minimum)  \n✅ **Focus States:** Clear blue outline on all interactive elements  \n✅ **Icon Accessibility:** All icons paired with text labels  \n✅ **Color Not Alone:** Status conveyed through color + icon + text  \n✅ **Responsive:** All layouts work at any screen size  \n✅ **Touch Targets:** Minimum 8x8px for interactive elements  \n\n---\n\n## Summary\n\nThe AI theme provides a cohesive, modern visual language across FleetNeuron with:\n\n1. **Consistency** through shared design tokens\n2. **Clarity** via clear hierarchy and status indicators\n3. **Polish** with glass morphism and glowing effects\n4. **Accessibility** with high contrast and clear labeling\n5. **Responsiveness** across all device sizes\n6. **Usability** with intuitive interaction patterns\n\nThis design system scales elegantly from HOS monitoring to financial invoicing, maintaining professional quality across all application sections.\n"
