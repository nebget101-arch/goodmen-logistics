### FN-329 (subtask)
**New file:** `backend/packages/goodmen-shared/services/investigation-pdf-service.js` (552 lines)

Two exported functions:

1. **`buildRequestPdf(driverData, employerData, oeData, publicLink)`** — generates outbound investigation request letter
   - OE header with company name, USDOT
   - From/To blocks with company details
   - Driver info (name, CDL, DOB, employment dates)
   - Standard FMCSA request language (6 required inquiry items per 49 CFR 391.23)
   - Online response link prominently displayed
   - Page footers with date and page numbers

2. **`buildResponsePdf(responseData, driverData, employerData, oeData)`** — generates completed response record
   - Same OE branding header
   - Driver/employer respondent sections
   - General (DOT testing), Accidents, Drug & Alcohol, Employment separation sections
   - Certification with legal text
   - Optional additional notes

**Patterns followed:** Uses pdf-lib (PDFDocument, StandardFonts, rgb), same helper pattern as pdf.service.js (drawSectionBanner, drawFieldPair, drawWrappedText, checkPage/newPageWithHeader)
