### FN-439 (subtask)

**Implementation Summary:**
Built 4-step toll import wizard replacing the placeholder component:
1. **Upload** — drag-and-drop/file picker for CSV, optional toll account selection
2. **Map** — column mapping with dropdown selects, load/save mapping profiles
3. **Preview** — table showing mapped data with row numbers
4. **Commit** — confirmation box + import button, progress indicator
5. **Result** — import statistics (imported, duplicates, errors, exceptions)

Added 4 service methods to `TollsService`:
- `uploadImportCSV()` — POST /import/upload
- `commitImport()` — POST /import/commit
- `getMappingProfiles()` — GET /import/mapping-profiles
- `saveMappingProfile()` — POST /import/mapping-profiles

Added model types: `TollUploadResult`, `TollCommitResult`, `TollMappingProfile`, `TOLL_NORMALIZED_FIELDS` (13 fields).

**Files Changed:**
