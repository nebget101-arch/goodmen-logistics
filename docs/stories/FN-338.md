# FN-338: Convert separate filter bars to inline table header filters across all data pages

## Requirement
The loads-dashboard uses inline filter controls in a second `<tr class="header-filters">` row inside `<thead>`. This pattern should be applied to all other data pages that currently use separate filter bars above their tables.

## Acceptance Criteria
- [x] All listed pages have inline filter controls inside a `<tr class="header-filters">` row in `<thead>`
- [x] Separate filter bars removed (replaced by inline header filters)
- [x] Filter functionality preserved (two-way binding, auto-refresh on change)
- [x] Consistent styling with loads-dashboard filter row
- [x] loads-dashboard raw `<select>` elements converted to `<app-ai-select>`
- [x] Dark theme styling consistent across all pages
