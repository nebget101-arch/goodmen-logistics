# FN-367: Replace Edit Stop popup modal with side panel during rate con review

## Summary
Replace the centered Edit Stop popup modal with an inline side panel that renders inside `.draft-review__right` during draft rate con review, keeping the PDF visible on the left.

## Acceptance Criteria
- [ ] Edit stop opens as an inline side panel (not a popup modal) during draft review
- [ ] Rate con PDF remains visible and scrollable while editing a stop
- [ ] All existing stop fields work in the side panel
- [ ] Save and Remove Stop actions work correctly from the side panel
- [ ] Side panel can be closed to return to the normal form view
- [ ] Works on screens 1280px and wider
