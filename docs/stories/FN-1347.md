# FN-1347: control-center.component.ts:220 references undeclared `widgets` (TS2663)

## Requirement
Render dev preview deploy for the FN-1327 PR fails to build because `frontend/src/app/components/control-center/control-center.component.ts:220` calls `matchPresetKey(widgets)` instead of `matchPresetKey(this.widgets)`. TypeScript flags it as `TS2663: Cannot find name 'widgets'` and the build aborts. Reviewers cannot validate the parent story (FN-1327, in Code Review under Epic FN-1321) until the preview is restored.

## Root Cause
Story FN-1327 added a private helper `matchPresetKey(widgets: readonly WidgetId[])` and three call sites in `ControlCenterComponent`. Two pass `this.widgets`; the third (inside `persistLayout()`, line 220) was written as a bare `widgets` identifier. The typo was introduced by subtask FN-1343 and merged into `integration/FN-1327`.

```ts
// integration/FN-1327 â€” control-center.component.ts

```
