# FN-409: Enhanced Import Preview UX -- AI Mapping Review

## Summary
Added an AI Analysis review step to the Fuel Import Wizard between file upload and
column mapping. The AI preprocessor analyzes uploaded files and suggests column
mappings with confidence scores, detects split strategies, and summarizes row
breakdown (normal/split/skipped/flagged).

## Changes

### `fuel-import-wizard.component.ts`
- Added `ai_analysis` wizard step between `upload` and `preview`
- New properties: `aiPreprocessData`, `aiLoading`, `aiError`, `showFlaggedRows`
- `runAiPreprocess()` -- calls `FuelService.aiPreprocess()` after file upload
- `acceptAiMapping()` -- copies AI column mapping to the wizard's manual mapping
  and advances to the preview step
- `skipAiMapping()` -- bypasses AI and goes to manual preview/mapping
- Helper methods: `getConfidenceClass()`, `getConfidenceLabel()`,
  `getSplitStrategyLabel()`, `aiMappingEntries` getter
- Graceful fallback: if AI call fails, shows error with "Map Manually" button
- Updated step numbering (8 steps total)
- Reset AI state in `startOver()`

### `fuel-import-wizard.component.html`
- New AI Analysis step panel (step 3) with:
  - Loading spinner during AI processing
  - Error banner with manual mapping fallback
  - Overall confidence banner (color-coded)
  - Column mapping table with confidence badges (green >= 80%, yellow >= 50%, red < 50%)
  - Split strategy chip
  - Row breakdown summary cards (Normal/Split/Skipped/Flagged)
  - Expandable flagged rows list
  - "Accept AI Mapping" (primary) and "Map Manually" (secondary) action buttons
- Updated step comments and back-navigation for all subsequent steps

### `fuel-import-wizard.component.css`
- Styles for AI confidence banner, mapping table, confidence badges,
  split strategy chip, row breakdown cards, flagged rows list
