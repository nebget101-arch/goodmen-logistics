# FN-521: Address autocomplete on employment application using SmartyStreets API

## Requirement
Add address autocomplete to the public employment application form so applicants can quickly fill in address fields using SmartyStreets API suggestions.

## Acceptance Criteria
- [ ] Typing a partial street address in Current Address shows SmartyStreets suggestions
- [ ] Typing a partial street address in Employer Address shows suggestions
- [ ] Selecting a suggestion auto-fills city, state, and zip
- [ ] Debounce works (no request fired on every keystroke)
- [ ] Manual entry still works if autocomplete is dismissed
- [ ] API key is NOT visible in browser network requests (proxied through backend)
- [ ] No console errors when autocomplete is used
