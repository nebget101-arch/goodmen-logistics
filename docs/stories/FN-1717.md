## Implementation Summary

### FN-1720 (subtask) — frontend
Re-engined the `/tracking` live map from Leaflet to **MapLibre GL** (`maplibre-gl@4.7.1`, lazy-loaded in `TrackingMapModule`):

- **3D camera:** MapLibre map over a dark, key-less CARTO `dark_all` raster basemap, default `pitch: 50°`, `bearing: -17.6°`, `maxPitch: 75°`. `NavigationControl({ visualizePitch: true })` + drag/touch rotate give the user free rotate / tilt / zoom.
- **Animated AI truck markers:** each vehicle is an HTML `maplibregl.Marker` (truck SVG glyph + pulsing radial-glow halo), status-tinted via CSS custom properties (moving = emerald, idle = amber, offline = slate, no pulse). Markers rotate to `heading_deg` (`rotationAlignment: 'map'`, `pitchAlignment: 'viewport'` so trucks stay upright/readable when the camera is tilted).
- **Smooth movement (no teleporting):** a single `requestAnimationFrame` tween loop, run **outside the Angular zone**, eases each marker from its current point to the new ping over ~900 ms. The map's own render loop is likewise created outside the zone, so 50+ trucks move at 60 fps without thrashing change detection.
- **Ported from the base map:** geofence overlay (now a GeoJSON `fill` + `line` source; circles approximated to a 64-gon ring; hover popup shows the geofence name), breadcrumb hover trail (GeoJSON `line` source), vehicle side panel, and the driver / status / geofence filter bar — all client-side filter + containment logic reused unchanged.
- **FN-317:** all `[options]` arrays remain plain readonly/reassigned class fields (asserted by spec).
- **Fail-soft:** map creation is wrapped in try/catch so WebGL-less contexts (headless CI) set a friendly error instead of throwing; the component still constructs.

## Key Decisions
- True 3D via MapLibre GL engine swap (chosen over faked 2.5D on Leaflet), per intake decision 2026-06-04.
- **`@types/maplibre-gl` not added.** `maplibre-gl` v4 ships its own bundled type definitions (`dist/maplibre-gl.d.ts`); the DefinitelyTyped `@types/maplibre-gl` package is a deprecated no-op stub and adding it risks duplicate-identifier errors. The AC's intent (TypeScript types available) is satisfied by the bundled types.
- **`skipLibCheck: true` enabled in `tsconfig.json`.** MapLibre's transitive type deps (`kdbush`, `@types/mapbox__point-geometry`) ship `.d.ts` authored for a newer TS lib (generic typed arrays, `export =`), which don't type-check under this repo's TS 5.2. `skipLibCheck` is the standard, low-risk fix (it skips checking declaration files only, not app code) and is recommended for Angular projects.
- **Key-less raster basemap (CARTO `dark_all`).** Avoids an API-key dependency. Delivers the dark theme + true 3D *camera*. Extruded 3D buildings (the AC's *optional* item) need a keyed vector-tile provider and are deferred — code is structured so a vector style can be dropped in later.
- **HTML markers over a symbol layer** for trucks: gives full CSS control of the glow/pulse + status tint and easy click/hover, and stays smooth at the AC's 50+ trucks target (tween loop runs outside the Angular zone).

## Deployment Handoff
- **Service to redeploy:** `fleetneuron-logistics-ui` (Angular frontend) — static rebuild picks up the new lazy `maplibre-gl` chunk. No backend changes.
- **New runtime dependency:** `maplibre-gl@^4.7.1` (bundled into the lazy `/tracking` chunk, ~810 KB; not added to the initial bundle). CI/CD must `npm ci` under **Node 20** (the lockfile was regenerated under Node 20).
- **No DB migrations, no env vars, no feature flags, no cache invalidation.** Reuses the existing `/api/vehicle-positions` endpoints + `vehicle:position` WS event (unchanged).
- **External tiles:** the map fetches CARTO basemap tiles (`*.basemaps.cartocdn.com`) at runtime — no key required, but the deploy environment must allow outbound HTTPS to that host (already permitted; same class as the prior OpenStreetMap tiles).
- **Post-deploy verification:** open `/tracking` → confirm dark 3D map renders tilted, trucks appear as glowing status-tinted icons, drag-rotate/tilt works, live pings glide (no teleport), geofence toggle + side panel + filters work.
