## Implementation Summary
Added `@openapi` JSDoc annotations to all 135 endpoints across 15 route files in `backend/packages/goodmen-shared/routes/`. Each annotation includes summary, description, tags, security scheme, parameters, request body schema, and response schemas (200, 400, 401, 404, 500) matching the actual handler code.

### Special Documentation Notes Applied
- **Public endpoints** (public-roadside, public-onboarding, public-consents): Marked with `security: []`
- **Safety/compliance endpoints**: Include regulatory references (49 CFR Parts 385, 390, 391, 395, 396)
- **DQF endpoints**: Document completeness calculation weights
- **Drug & alcohol endpoints**: Note FMCSA Clearinghouse integration (49 CFR Part 382)
- **HOS endpoints**: Reference FMCSA hours-of-service rules (49 CFR Part 395)
