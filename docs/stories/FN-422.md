### FN-447 (subtask) — BE: Add toll consumption logic

**Changes in `settlement-service.js`:**
- Added `consumeTollsForSettlement()` — queries unlinked toll transactions for driver/period, checks `expense_responsibility_profiles.toll_responsibility`
  - `company` -> skip, no deduction
  - `driver` -> full deduction (100%)
  - `shared` -> applies `custom_rules.toll_split_percentage` (defaults to 50%)
- Creates `settlement_adjustment_items` with `source_type = 'imported_toll'`
- Links toll transactions (sets `settlement_id`, `settlement_adjustment_item_id`, `settlement_link_status = 'linked'`)
- Integrated into `createDraftSettlement()` after recurring deductions, before recalc

**Changes in `tolls.js`:**
- `POST /api/tolls/transactions/:id/post-to-settlement` — manual toll-to-settlement linking
- `POST /api/tolls/transactions/:id/unlink-from-settlement` — reverse operation with approved/void guards
