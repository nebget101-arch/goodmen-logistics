## Implementation Summary
All five components plus the `.ai-panel-flat` utility were built in a single
frontend pass (FN-1638). They are pure-additive: only new files under
`frontend/src/app/shared/**` and `frontend/src/app/dev/**`, plus append-only
edits to `SharedModule`, `AppModule`, the router, `styles.css`, and the theme
reference doc. No existing page template/component was touched.

Each primitive is `OnPush` and self-documents its contract in a co-located
`README.md`. Pure logic (aria-label composition, dominant-severity, length
normalization, severity labels) is exposed as static helpers so specs cover it
without TestBed; DOM/event behavior is covered with TestBed specs.

### FN-1638 (subtask)
- **kpi-card** — `label/value/subline/status/trend/routerLink`; renders as `<a>`
  when `routerLink` is set, else `<div role="group">`. 3px left accent strip +
  hover lift/glow tinted by status. `aria-label` composed via
  `KpiCardComponent.composeAriaLabel`.
- **ai-segmented-control** — `segments/selectedKey` + `selectedKeyChange`
  (two-way). Active = flat `.btn-primary` gradient (no glow), inactive
  `.btn-secondary`; `aria-pressed` per button, `role="group"` container.
- **ai-skeleton** — `width/height/radius` (numbers → px via
  `AiSkeletonComponent.toCssLength`). Host `role="status"`, `aria-busy="true"`,
  `aria-label="Loading"`; shimmer disabled under `prefers-reduced-motion`.
- **ai-hero-strip** — `items` (≤3 chips); dominant severity from
  `critical > warning > info`, empty → green "All systems nominal". `[heroAction]`
  content slot. Built on `.ai-panel-flat`.
- **ai-alert-row** — `severity/category/message/date/routerLink?` + `acknowledge`
  / `snooze` outputs. Two `.btn-icon` actions; severity pip + visually-hidden
  `.sr-only` label (color never alone).
- **.ai-panel-flat** — appended to `frontend/src/styles.css`; no shadow, 3px top
  accent border keyed off `data-severity`.
- **Dev showcase** — `DashboardPrimitivesComponent` at `/dev/dashboard-primitives`
  (route spread-gated by `!environment.production`).
- **No new hex colors** — verified: `grep -rEn "#[0-9a-fA-F]{3,8}"` over all new
  folders returns nothing; colors are `rgba()` from the documented palette.

## Key Decisions
- **SharedModule, not AppModule.** The ticket says "same pattern as
  `app-ai-select`/`app-ai-date-picker`" — those live in `SharedModule`, so the
  primitives go there too (and the story AC explicitly allows the shared-module
  path). `RouterModule` was added to `SharedModule` imports because three
  primitives use `routerLink`.
- **No new hex.** The grep gate only flags `#hex`, but the intent is "reuse the
  palette". Colors are expressed as `rgba()` with the exact channel values
  already documented in the visual reference, so the gate passes and no new
  color is introduced.
- **Dev route via array spread.** `...(environment.production ? [] : [route])`
  keeps the route out of production bundles without a guard, and the file
  replacement (`environment.prod.ts`) drives the gate.
- **Static helpers for logic.** Pure functions (`composeAriaLabel`,
  `dominantSeverity`, `toCssLength`) make the AC-mandated specs fast and
  TestBed-free where possible.
