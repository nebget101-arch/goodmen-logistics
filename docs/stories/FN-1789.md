# FN-1789: Adapter — Equipment / Motor-Carrier Lease Agreement signing

> Epic: **FN-1786**. Blocked by **FN-1787 + FN-1788** (uses the generic engine).

## Requirement
Wire the upload→scan→fill→sign engine into the **Equipment / Motor-Carrier Lease Agreement** workflow. From a leased unit / equipment owner (`vehicles.ownership_type='leased'`, `equipment-owners` area), an internal user uploads the lease PDF, prepares it via the engine, and sends it to the **equipment owner / lessor** to e-sign. The signed agreement links back to the vehicle/equipment-owner record.

- [ ] Backend links a completed signature request to the vehicle/equipment-owner (request id + signed-PDF key).
- [ ] Internal user sees signing status (sent/viewed/signed) + downloads signed PDF from the equipment record.
- [ ] Reuses generic engine endpoints — no duplicate signing logic.
