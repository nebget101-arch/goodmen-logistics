# FN-1323: KPI strip + global Today / 7d / 30d window selector with deltas

## Requirement
Replace the existing fleet-health metric tiles with a compact top-of-page KPI strip showing targets, deltas vs. previous window, and a global window selector that scopes every card on the page.

## Acceptance Criteria
- [ ] Top-of-page KPI strip with at most 6 KPIs visible at once.
- [ ] Each KPI shows: current value, target (if defined), delta vs. previous window, colored indicator (good / warn / critical).
- [ ] Global window selector (Today / 7d / 30d) lives next to the page header; changing it re-fetches every KPI scoped to the chosen window.
- [ ] Selected window persists in URL (`?window=`) and is restored on reload.
- [ ] All KPI cards remain click-throughs to detail pages.

## API Contract (defined here so frontend + backend can run in parallel)
```
GET /api/dashboard/stats?window=today|7d|30d
→ 200 {
    window: "today|7d|30d",
    current:  { activeDrivers, totalDrivers, hosViolations, dqfComplianceRate, ... },
    previous: { ... same shape ... },
    delta:    { activeDrivers: +3, hosViolations: -2, ... }
  }
```
Window math is timezone-aware (operating-entity timezone). When `window` is omitted, behavior is unchanged from today.

## Implementation Summary

### FN-1332 (subtask)
Frontend slice of the KPI strip + window selector. Adds:

- **`DashboardWindowService`** (`frontend/src/app/services/dashboard-window.service.ts`) — single source of truth for the active dashboard window. URL-backed: reads `?window=` on init, listens to `NavigationEnd` for back/forward, and `setWindow()` writes the URL via `Router.navigate(..., { queryParamsHandling: 'merge', replaceUrl: true })`. Default is `7d`. Unknown values are ignored.
- **`WindowSelectorComponent`** (`control-center/window-selector/`) — pill-segmented Today / 7d / 30d radio group rendered in the Control Center header. Standalone, OnPush, ARIA `role="radiogroup"`.
- **`KpiStripComponent`** (`control-center/kpi-strip/`) — six click-through KPI cards (Active Drivers, Vehicles OOS, HOS Violations, DQF Compliance, Medical Certs, Maintenance Due). Re-fetches on every window change. Renders deltas with good/bad/neutral tone driven by per-KPI polarity (`higher-better` vs `lower-better`); DQF deltas are shown as percentage points. Gracefully handles **both** the legacy flat-stats response and the new `{current, previous, delta}` envelope from FN-1333, so frontend and backend can land in any order.
- **`ApiService.getDashboardStats({ window? })`** — accepts an optional `window` param. When omitted, the endpoint behaves exactly as before.
- **`ControlCenterComponent`** mounts the window selector in its header and the KPI strip directly below.
- **`DashboardComponent.html`** — old `panel panel-metrics` block (the previous fleet-health 8-tile grid) is removed; the KPI strip replaces it via the Control Center.

## Key Decisions

- **URL is the source of truth for the window**, not the service. The service mirrors the URL so a page reload, deep link, or back/forward press always restores the user's chosen window without local-storage divergence.
- **Tolerant response parser** in the KPI strip. The component accepts the legacy flat shape (current behavior of `dashboard-stats-service`) and the new `{current, previous, delta}` envelope (FN-1333). When `delta` is absent, deltas simply don't render — no error state. This decouples FN-1332 from FN-1333.
- **6 KPIs, not 8**, per the AC ("at most 6 KPIs visible at once"). Dropped Active Loads (already covered by the Loads progress panel below) and Clearinghouse (low-frequency signal) from the original 8-tile grid; kept Vehicles OOS over Vehicle Status because OOS is the action signal.
- **Per-KPI delta polarity**. Each KPI declares whether higher or lower is better; the renderer uses that to color the delta good/bad/neutral. DQF specifically uses percentage-point formatting (`+3pp`) since the underlying value is already a percent.
- **Severity color comes from the *current value*, not the delta**. A 0% improvement on a 50% DQF compliance rate is still critical — the delta tone and severity tone are independent.
