## Key Decisions
1. **Output-only component** ‚Äî hero CTA emits typed events; all modal state lives in `loads-dashboard` so no duplication of existing bulk/auto-create logic.
2. **`_handleFiles` single/bulk split** ‚Äî `pdfs.length === 1` ‚Üí single, else bulk. Sliced to 10 max at component level before emit.
3. **Drag-drop on the card, not just the inner zone** ‚Äî the entire upload card is a drop target for a larger hit area; visual feedback (`hero-card--drag-over`) applied at card level.
4. **`openCloneLoad()` stub** ‚Äî avoids a dead CTA; shows a dismissing error toast until the clone backend lands.
5. **Stubbed `tenantInboundEmail`** ‚Äî constant in the component; replaced by API call when FN-726 (email inbound) lands.
