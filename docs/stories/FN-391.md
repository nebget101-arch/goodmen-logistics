### FN-396 (subtask) — Make attachment badges downloadable links

**Changes:**
- Replaced the single clickable `div` wrapper with individual click handlers per `<app-attachment-chip>`
- Added `downloadAttachmentByType(load, type)` method that fetches the load detail, finds the first attachment matching the type, and opens its URL in a new tab via `window.open()`
- Added `.attachment-chip--download` CSS class with cursor pointer and underline-on-hover for download affordance
- Each badge now has `$event.stopPropagation()` so clicking a specific badge downloads that file directly

**Files Changed:**
- `frontend/src/app/components/loads-dashboard/loads-dashboard.component.html` — individual click handlers per chip
- `frontend/src/app/components/loads-dashboard/loads-dashboard.component.ts` — `downloadAttachmentByType()` method
- `frontend/src/app/components/loads-dashboard/loads-dashboard.component.scss` — `.attachment-chip--download` styles
