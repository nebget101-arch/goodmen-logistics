### FN-1644 (subtask) — Implementation notes

Integrated into `integration/FN-1642` at commit `3a1b2c9d` (fast-forward from base `f335d97c`).

**Components (under `frontend/src/app/shared/`, rgba-only — zero hex literals):**

- **`<app-status-stepper>`** (`status-stepper/`) — Inputs `steps: Array<{key,label,kicker?,value?,status}>`, `activeKey`; Output `stepChange`. States: pending (slate, clickable), current (cyan border + glow, `aria-current="step"`), complete (green ✓), skipped (line-through), blocked (dashed border, `aria-disabled`, non-focusable). Roving-tabindex keyboard nav: Arrow/Home/End move focus across reachable steps, Enter/Space activate.
- **`<app-summary-chip>`** (`summary-chip/`) — Inputs `title`, `detail`, `editable`; Output `edit`. Green left tick + bold title + ellipsised detail + optional pencil button.
- **`<app-side-rail-tabs>`** (`side-rail-tabs/`) — Inputs `tabs: Array<{key,label,icon?}>`, `activeKey`; Output `activeKeyChange` (`[(activeKey)]`-friendly). Underlined cyan-active tab strip; `role="tablist"`/`role="tab"` + `aria-selected`. Content via `ng-content` slotted with `[data-rail-pane="<key>"]`; component toggles pane visibility.
- **`ToastService` + `<app-toast-host>`** (`toast/`) — `providedIn:'root'`; `success/error/info` push toasts (4s auto-dismiss, manual ✕ close) via `BehaviorSubject`. Host mounted once at app root in `app.component.html`. **Audit result: no pre-existing shared ToastService → built fresh.**

**CSS utilities (appended to `frontend/src/styles.css`):** `.num-field` (right-aligned monospace input + unit chip for min/%/$) and `.read-field` (dashed read-only display + 28px `.btn-icon` pencil).

**Wiring:** components + host declared/exported in `SharedModule` (the real `app-ai-select`/`app-ai-date-picker` pattern; ticket text said `AppModule`). Dev page `RoadsidePrimitivesComponent` declared in `AppModule`. Dev-only route `dev/roadside-primitives` pushed to the routes array only when `!environment.production`.

**Docs:** appended "Roadside primitives" section (contracts, status→color map, stepper state matrix) to `docs/AI_THEME_VISUAL_REFERENCE.md`.

**Tests:** 25 unit specs (stepper kbd nav + aria-current; summary-chip edit emit; side-rail-tabs activeKeyChange + pane visibility; toast `fakeAsync`/`tick(4000)` auto-dismiss). A karma `test` target + `src/test.ts` + `tsconfig.spec.json` were added (none existed). Specs verified via `tsc -p tsconfig.spec.json` (0 errors); **karma/jasmine runtime devDependencies must be installed to execute them headlessly.**

**Verification:** `ng build --configuration development` → exit 0. No-hex grep over the 4 new shared folders → 0 matches. No build artifacts committed.
