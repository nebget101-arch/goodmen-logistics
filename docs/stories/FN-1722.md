# FN-1722: Story K — Live map polish: US styling, street zoom, follow-unit, open-in-Maps

## Requirement
Polish the MapLibre live map (FN-1717) for real-world demo use. It currently uses the minimal CARTO `dark_all` basemap — no state borders, blurry when zoomed, no per-unit focus. Make it US-legible, street-level capable, and interactive.

## Acceptance Criteria
- [ ] US state borders + place labels render (map reads as a US map; keep AI dark theme — e.g. `dark_matter` *with labels* or an admin-boundary vector layer). No empty dark void.
- [ ] Zooming in resolves to crisp street detail (raise `maxZoom`; street-level tile source).
- [ ] Follow-the-unit: clicking a vehicle eases the camera to it and re-centers on each new ping until deselected/panned, with a clear exit affordance.
- [ ] Vehicle side panel shows lat/lng + an "Open in Google Maps" link (`https://www.google.com/maps?q=<lat>,<lng>`, new tab, `rel="noopener"`).
- [ ] AI dark-theme tokens only; `[options]` = readonly class fields (FN-317); geofence overlay + filters preserved.
