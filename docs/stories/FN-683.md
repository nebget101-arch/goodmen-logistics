## Implementation Notes

### FN-707 (subtask)
**Work Order parts issuance — show read-only bin from inventory.bin_id**

Added a "Bin" column to the Parts & Labor table in the work order component:
- New column header + cell between "SKU" and "Quantity"
- `resolveBinDisplay()` method resolves bin info from 3 sources (in priority order):
  1. `bin` object joined from `location_bins` via `bin_id` → shows `"A-1 (Shelf A)"`
  2. Flat `bin_code`/`bin_name` fields → same display format
  3. Legacy `bin_location` free-text → shows as-is
  4. Falls back to "—" dash if no bin data
- Populated automatically when a part is looked up via `onPartLookup()`
- Read-only display (not editable)
- Styled with blue text for visibility

**Files:**
- `frontend/src/app/components/work-order/work-order.component.html` — added Bin column header + cell
- `frontend/src/app/components/work-order/work-order.component.ts` — added `resolveBinDisplay()` method, populated in `onPartLookup()`
- `frontend/src/app/components/work-order/work-order.component.css` — added `.bin-readout` styles
