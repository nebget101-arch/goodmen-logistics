## Implementation Notes

### FN-704 (subtask)
**Reusable `<bin-picker>` component with inline create + text fallback**

**Branch:** `frontend/FN-704/bin-picker-component`

1. **`frontend/src/app/shared/components/bin-picker/bin-picker.component.ts`** (new):
   - `@Input() locationId` — location context; triggers `api.getLocationBins()` on init/change
   - `@Input() value: string | null` — selected `bin_id`; two-way binding partner for `[(value)]`
   - `@Input() allowCreate = false` — when true, `+ Create new bin…` sentinel option appears
   - `@Input() fallbackToText = true` — renders text input when location has no bins
   - `@Input() placeholder`, `label`, `disabled`
   - `@Output() valueChange`, `@Output() create`
   - `CREATE_SENTINEL = '__create__'` constant guards the inline-create trigger
   - `formatBin(bin)` → `"A-1 (bin_name)"` or `"A-1 (zone, aisle, shelf, position)"` or `"A-1"`
   - `onSelectChange()` resets visual selection before opening create dialog (sentinel doesn't stick)
   - `saveNewBin()` POSTs to `api.createLocationBin()`, auto-selects new bin, emits `valueChange` + `create`
   - Handles both array response and `{ data: [] }` wrapped shapes from the bins API
   - OnPush change detection with explicit `cdr.markForCheck()` after all async ops

2. **`frontend/src/app/shared/components/bin-picker/bin-picker.component.html`** (new):
   - Loading spinner state (`*ngIf="loading"`)
   - Error banner state (`*ngIf="loadError"`)
   - Fallback text input mode (`*ngIf="!loading && !loadError && showFallback"`) with "No bins configured. Add bins in **Admin → Locations**" note
   - Native `<select>` mode (`*ngIf="!loading && !loadError && !showFallback"`) with placeholder + create option + bin `*ngFor`
   - Inline create dialog overlay — `bp-backdrop` → `bp-dialog`, fields: `bin_code` (required), `bin_name`, `bin_type` dropdown, `zone`; spinner on save; error message

3. **`frontend/src/app/shared/components/bin-picker/bin-picker.component.scss`** (new):
   - `.bp-*` CSS namespace
   - Dark fleet theme: `rgba(15, 23, 42, 0.98)` backgrounds, `#38bdf8` accents, backdrop blur
   - Consistent with existing location dialog modals
   - Custom `<select>` chevron via SVG background-image

4. **`frontend/src/app/shared/shared.module.ts`** (modified):
   - Declared `BinPickerComponent`
   - Exported `BinPickerComponent` so all feature modules that import `SharedModule` can use `<app-bin-picker>`
