### FN-335 (subtask)
Added 4 enhancements to the investigation-panel component:

1. **Download Response button** — appears when employer has a response with a document. Uses existing DQF `/documents/:id/download` endpoint via blob download.
2. **Inline response summary** — shows response type and received via in a green-tinted card when response exists.
3. **Email timestamps** — shows inquiry sent, follow-up sent, and response received timestamps with icons.
4. **New response badge** — green pulsing "NEW" badge on employer cards with responses received within the last 3 days.

**Files changed:**
- `frontend/src/app/services/employer-investigation.service.ts` — added `EmployerResponse` interface, `responses` field to `PastEmployerInvestigation`, response mapping, download method
- `frontend/src/app/components/drivers/investigation-panel/investigation-panel.component.ts` — added download, timestamp formatting, response helpers
- `frontend/src/app/components/drivers/investigation-panel/investigation-panel.component.html` — added timestamps, response summary, download button, new badge
- `frontend/src/app/components/drivers/investigation-panel/investigation-panel.component.css` — styles for all new elements
