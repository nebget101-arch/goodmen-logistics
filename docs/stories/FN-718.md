## Implementation Summary

### Entity Cache Service
- `EntityCacheService` â€” singleton service with `BehaviorSubject` per entity (customers, vehicles, parts)
- 5-minute TTL with auto-refresh on expired access
- `addRecentlyUsed()` / `getRecentlyUsed()` backed by localStorage (max 10 per entity)
- `invalidateCache()` for manual refresh

### Smart Autocomplete Component
- `SmartAutocompleteComponent` â€” reusable, `OnPush` change detection
- Inputs: `items`, `displayField`, `searchFields`, `placeholder`, `value`, `maxResults`, `recentItems`, `itemTemplate`, `loading`
- 150ms debounced local search across configurable fields
- Keyboard navigation (arrow up/down, Enter to select, Escape to close)
- Click-outside-to-close via `@HostListener`
- "Recently Used" section header + divider
- Truncation note when results exceed `maxResults`
- Dark theme CSS with smooth transitions
