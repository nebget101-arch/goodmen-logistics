### FN-428 (subtask) — Transactions Data Grid

**Changes:**
- Added `TollTransaction` interface to `tolls.model.ts` with all transaction fields
- Added `getTransactions()` and `createTransaction()` methods to `tolls.service.ts`
- Replaced placeholder with full data grid: Date, Provider, Plaza/Location, Truck, Driver, Load#, Amount, Status columns
- Sortable by Date (default DESC) and Amount with toggle direction and sort icon
- Filterable by date range (from/to pickers), matched status (dropdown)
- Pagination: 50 per page with page controls and range display
- Color-coded status badges (matched=green, unmatched=red, partial=amber, manual=blue)
- Added `FormsModule` to `TollsModule` for ngModel bindings
