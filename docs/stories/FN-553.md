# FN-553: Rename "Primary Payee" / "Additional Payee" to "Driver" / "Equipment Owner" in Pay Routing UI

## Requirement
Throughout the Pay Routing and Scheduled Deduction UI, rename display labels for clarity:

- "Primary Payee" → "Driver"
- "Additional Payee" → "Equipment Owner"
- "Primary + Additional payee" badge → "Driver + Equipment Owner"

**Backend field names and DB values remain unchanged** (`apply_to: 'primary_payee'`, `additional_payee_id`, etc.) — only display text changes.

## Acceptance Criteria
- [x] All instances of "Primary Payee" in `dispatch-drivers.component.html` replaced with "Driver"
- [x] All instances of "Additional Payee" replaced with "Equipment Owner"
- [x] Apply To dropdown in Scheduled Deduction dialog shows "Driver" and "Equipment Owner" (not "Primary payee" / "Additional payee")
- [x] Pay Routing section shows "Driver" and "Equipment Owner" labels
- [x] `settlement-detail.component.html` — "Primary payee" / "Additional payee" label rows renamed
- [x] Any other settlement list/shared component templates showing payee type labels updated
- [x] Backend `apply_to` values ('primary_payee', 'additional_payee') NOT changed in DB or API

## Implementation Summary

### Changes Made

**`dispatch-drivers.component.ts`**
- `deductionTargetOptions`: `'Primary payee'` → `'Driver'`; `'Additional payee'` → `'Equipment Owner'`
- `getSuggestedTargetsLabel()`: `'Primary + Additional payee'` → `'Driver + Equipment Owner'`; `'Additional payee only'` → `'Equipment Owner only'`; `'Primary payee only'` → `'Driver only'`
- `getRecurringDeductionTargetLabel()`: `'Additional payee'` → `'Equipment Owner'`; `'Primary payee'` → `'Driver'`

**`dispatch-drivers.component.html`**
- Pay Routing section: `"Primary payable to"` label → `"Driver"`

**`settlement-detail.component.html`**
- Summary section: `"Primary payee"` → `"Driver"`, `"Additional payee"` → `"Equipment Owner"`
- Totals section: `"Additional payee subtotal"` → `"Equipment Owner subtotal"`, `"Net additional payee"` → `"Net equipment owner"`
- Loads table header: `"Additional payee"` → `"Equipment Owner"`
- Email options: `"Send to additional payee"` → `"Send to Equipment Owner"`

**`equipment-owners.component.html`**
- Billing panel: `"Additional payee"` checkbox label → `"Equipment Owner"`
- `"Additional payee rate, %"` → `"Equipment Owner rate, %"`
- `"Active (visible in Additional Payee search)"` → `"Active (visible in Equipment Owner search)"`

### Key Decision
Only display labels changed. All Angular `[(ngModel)]` bindings, API field names (`apply_to: 'primary_payee'`, `additional_payee_id`), and DB column names are left untouched.
