## Implementation Summary

### FN-1707 (subtask — database)

Created `20260610120000_create_user_sessions.js` in `backend/packages/goodmen-database/migrations/`.

**Schema:**
- `id` uuid PK via `gen_random_uuid()` (pgcrypto extension)
- `user_id` uuid FK → `users(id)` ON DELETE CASCADE
- `tenant_id` uuid FK → `tenants(id)` ON DELETE SET NULL, nullable
- `session_token_hash` varchar(128) UNIQUE NOT NULL — SHA-256 of JWT secret; never stores raw token (mirrors `password_reset_tokens` pattern)
- `user_agent` varchar(512) nullable
- `ip_address` varchar(45) nullable (IPv6-safe)
- `device_label` varchar(255) nullable
- `created_at` timestamp DEFAULT now()
- `last_seen_at` timestamp DEFAULT now()
- `revoked_at` timestamp nullable
- `revoked_reason` varchar(100) nullable

**Indexes:**
- `idx_user_sessions_user_revoked` ON `(user_id, revoked_at)` — active-sessions-for-user lookups
- `idx_user_sessions_tenant_id` ON `(tenant_id)` — tenant-scoped lookups
- `session_token_hash` UNIQUE constraint (inline)

**`down`:** drops both indexes then drops table.
