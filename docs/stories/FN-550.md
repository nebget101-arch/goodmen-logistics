# FN-550: Add Equipment Owner Percentage to Pay Configuration

## Requirement
Add Equipment Owner % and Company Retains % fields to the driver Pay Configuration section so fleet accountants can configure how revenue is split between Driver, Equipment Owner, and Company.

## Acceptance Criteria
- [ ] Three rows when pay model = percentage: Driver % (editable), Equipment Owner % (new editable), Company Retains (read-only = 100 - Driver% - EO%)
- [ ] Owner Operator: hide EO%, show only Driver % and Company Retains
- [ ] Inline validation error if Driver% + EO% > 100
- [ ] Save: include equipment_owner_percentage in compensation profile payload
- [ ] Load: populate from compensation_profile.equipment_owner_percentage
