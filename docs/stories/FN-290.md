# FN-290: Send Application Link button on DQF page with email dialog and status tracking

## Requirement
Add a "Send Application Link" button to the Drivers/DQF page that opens a modal for admin/safety users to invite drivers by email. The modal collects name and email, calls the invite endpoint, and displays success/error feedback.

## Acceptance Criteria
- [ ] "Send Application Link" button visible to admin/safety roles only
- [ ] Modal with First Name, Last Name, Email (required), Phone (optional)
- [ ] Form validation (required fields, email format)
- [ ] Submit calls POST /api/onboarding/invite
- [ ] Loading spinner during API call
- [ ] Success message: "Application link sent to {email}"
- [ ] Error display for duplicates/failures
- [ ] Dark theme styling consistent with FleetNeuron

## Key Decisions
- Used overlay modal pattern (not inline form) for invite to keep it separate from Add Driver flow
- Email sent fire-and-forget by backend; frontend shows success immediately on 201 response
- Refreshes driver list after successful invite to show new pending driver
