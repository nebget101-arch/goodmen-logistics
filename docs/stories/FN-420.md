### FN-440 (subtask) — AI: Claude Vision toll invoice extraction handler

**Changes:**
- Created `toll-invoice-vision-handler.js` in ai-service handlers
- Accepts base64 image + media type, sends to Claude Vision API with structured extraction prompt
- Extracts: invoice metadata (date, provider, plate, total, late fees) + array of individual toll transactions
- Each transaction includes: date, provider, plaza, entry/exit location, city, state, amount, reference ID
- Handles common formats: EZPass, SunPass, FasTrak, Illinois Tollway, Indiana Toll Road, Kansas Turnpike, Pay-By-Plate MA
- Multi-toll invoices return array of extracted line items
- Confidence score and warnings for extraction uncertainties
- Registered route: `POST /tolls/invoice-vision`

**Files Changed:**
- `backend/microservices/ai-service/src/handlers/toll-invoice-vision-handler.js` (NEW)
- `backend/microservices/ai-service/src/ai-router.js` — registered new route
