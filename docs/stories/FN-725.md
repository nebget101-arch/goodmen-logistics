## Implementation Notes

### FN-733 (subtask)
**Step 1 — Basics tab (load number, broker, rate, PO, status, notes)**

Created `StepBasicsComponent` with:
- Load Number (auto-generated `LD-YYMMDD-NNNN`, editable)
- Status dropdown (BOOKED → CANCELED)
- Billing Status dropdown (UNBILLED → VOID)
- Broker autocomplete with search/filter/select + clear button
- "Create New Broker" quick-action modal (calls `LoadsService.createBroker()`)
- PO Number input
- Rate input (currency, formatted on blur)
- Dispatcher (read-only, auto-populated from localStorage user)
- Collapsible Notes textarea
- AI-prefilled fields show sparkle icon (✨) via `aiPrefilledFields` input Set
- Required field validation (loadNumber, status, billingStatus) with inline errors
- `@Output() valid` and `@Output() dataChange` for parent wizard integration
