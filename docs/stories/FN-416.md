# FN-416: Build Toll Accounts & Devices Create/Edit Dialogs

## Sub-task: FN-425 -- Device Create/Edit Dialog with Truck/Driver Autocomplete

### What was done
- Created `DeviceDialogComponent` at `frontend/src/app/tolls/tolls-devices/device-dialog/`
- Dialog form fields: Toll Account (dropdown), Device Number, Plate Number, Truck (text autocomplete), Driver (text autocomplete), Effective Start Date, Effective End Date, Status (edit only)
- Truck autocomplete loads vehicles from `ApiService.getVehicles()` and filters by unit number
- Driver autocomplete loads drivers from `ApiService.getDrivers()` and filters by name
- Validation: Toll Account is required
- Calls `TollsService.createDevice()` / `updateDevice()` for save
- Success/error messages displayed after save; device list refreshes on success
- Added "Add Device" button to devices tab header
- Added "Edit" button on each device row
- Updated `TollsModule` to import `ReactiveFormsModule`, `FormsModule`, `MatDialogModule`
- Added `dark-dialog` global panel class in `styles.css` for dark-themed Material dialog
