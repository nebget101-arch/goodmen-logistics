## Scope

- New `ActionQueueComponent` (standalone, OnPush) at `frontend/src/app/components/dashboard/action-queue/`.
- Wired into `ControlCenterComponent` widget grid in place of the deleted `SmartAlertsComponent`.
- Right-rail Compliance Alerts feed removed from `DashboardComponent` (file + state + helpers + CSS).
- New `ApiService.getActionQueue()` and `ApiService.dismissActionQueueGroup()` consume the FN-1330 endpoints; legacy `getAlerts()` removed.
- Layout-store widget id `smart-alerts` renamed to `action-queue` (frontend `WidgetId`, role defaults; backend `ROLE_DEFAULTS`); legacy id auto-migrated by `sanitizeLayout`.
- Orphaned `SmartAlertsService` deleted.
