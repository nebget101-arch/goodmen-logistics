- [ ] Cache hit returns immediately with no OpenAI API call

## Implementation Summary

### FN-738 — Confidence Scoring (branch: `ai/FN-738/confidence-scoring`)

Enhanced `backend/packages/goodmen-shared/services/load-ai-extractor.js`:

**Threshold constants:**
- `CONFIDENCE_TIER_GREEN = 0.95` (>95% → green)
- `CONFIDENCE_TIER_YELLOW = 0.80` (80–95% → yellow; <80% → red)
- `AUTO_APPROVE_THRESHOLD = 0.90`

**New helpers:**
- `confidenceTier(score)` → `'green' | 'yellow' | 'red'`
- `computeOverallConfidence(conf)` → `Math.min(...requiredFields)` (broker, rate, pickup, delivery)

**Success return now includes:**
```js
{
  confidence:              { brokerName, poNumber, rate, pickup, delivery },  // 0–1 per field
  confidence_tiers:        { brokerName, poNumber, rate, pickup, delivery },  // 'green'|'yellow'|'red'
  overall_confidence:      0.0–1.0,   // min of the 4 required fields
  overall_confidence_tier: 'green'|'yellow'|'red',
  auto_approve:            true|false  // mode==='bulk' && overall >= 0.90
}
```

**Error returns** (no API key, PDF text failure, empty text, garbled text) all use a `ZERO_CONFIDENCE` spread so callers always get a consistent response shape.

**Exported:** `confidenceTier`, `computeOverallConfidence`, `AUTO_APPROVE_THRESHOLD`, `CONFIDENCE_TIER_GREEN`, `CONFIDENCE_TIER_YELLOW`

**`opts` parameter:** `extractLoadFromPdf(buffer, filename, opts = {})` — `opts.mode` drives `auto_approve` logic (`'single'` default, `'bulk'` for batch upload)

## Key Decisions
1. **`Math.min` for overall_confidence** — weakest required field drives the overall score; a high-confidence broker name doesn't compensate for a zero-confidence rate.
2. **`ZERO_CONFIDENCE` spread** — all error early-returns use a single const so callers always receive a consistent object shape regardless of failure mode.
3. **`opts.mode` parameter** — adding `mode` to `opts` (not the top-level signature) is non-breaking; all existing callers pass `(buffer, filename)` and get `auto_approve: false` by default.
4. **Exported helpers** — `confidenceTier` and `computeOverallConfidence` exported so consumers (frontend display logic, tests) can apply the same thresholds without duplicating them.

## Deployment Handoff
- **Services to restart**: `goodmen-shared` is a shared package; any microservice that requires it (ai-service, loads-service, bulk-upload-service) will pick up changes on their next deploy.
- **Migrations**: None (FN-741 adds the cache table — separate subtask).
- **Env/config changes**: None.
- **Feature flags**: None — confidence scoring and `auto_approve` are additive; existing callers ignore new fields.
- **Deploy order**: Backend / shared package only; restart ai-service and loads-service after merge.
- **Post-deploy verification**:
  1. Upload a clear PDF → confirm response includes `confidence_tiers`, `overall_confidence`, `overall_confidence_tier`, `auto_approve: false` (single mode).
  2. Call with `opts.mode = 'bulk'` and a high-quality PDF → confirm `auto_approve: true` when `overall_confidence >= 0.90`.
  3. Upload a garbled/empty PDF → confirm `auto_approve: false` and `overall_confidence: 0` in the error shape.

### FN-739 — Vision Fallback (branch: `ai/FN-739/vision-fallback`)

Added to `backend/packages/goodmen-shared/services/load-ai-extractor.js`:

- **`isScannedPdf(text)`** — heuristic: text < 100 chars OR alphanumeric ratio < 20% → scanned
- **`extractWithVision(buffer, filename)`** — Claude Vision fallback:
  - Converts PDF pages to PNG via `pdftoppm` (Poppler, 200 DPI, max 5 pages)
  - Calls Anthropic Claude Vision API (`ANTHROPIC_API_KEY`, model `ANTHROPIC_VISION_MODEL`) with same prompt
  - 90-second timeout, full temp file cleanup
- Modified `extractLoadFromPdf()`: after text extraction, if `isScannedPdf() || looksGarbled() || !pdfText` → vision fallback before giving up
- Vision path returns `extraction_method: 'vision'` + `vision_fallback_reason`
- Text path returns `extraction_method: 'text'`

**Env vars added**: `ANTHROPIC_API_KEY` (vision), `ANTHROPIC_VISION_MODEL` (optional, defaults to `claude-sonnet-4-20250514`)
**System dep**: `pdftoppm` (Poppler — already available for pdftotext)
