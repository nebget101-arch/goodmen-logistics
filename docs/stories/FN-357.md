# FN-357: Redesign Auto-Create from PDF and Bulk Upload modals with AI-themed design

## Requirement
The Auto-Create from PDF (single) and Bulk Upload Rate Confirmations modals on the Loads Dashboard are key AI features but currently use plain, generic styling. Redesign both modals with the FleetNeuron AI theme: glass morphism panels, AI icons (auto_awesome, neurology), cyan accents, gradient buttons, and animated processing states.

## Acceptance Criteria
- [ ] Auto-Create from PDF modal has AI-themed header with sparkle/brain icon
- [ ] Auto-Create modal has enhanced dropzone with AI branding and drag animation
- [ ] "Run Extraction" button uses gradient AI button style
- [ ] Extraction processing state shows AI-themed animation
- [ ] Bulk Upload modal has AI-themed header with sparkle icon
- [ ] Bulk Upload file list has polished styling with status indicators
- [ ] "Upload & Extract" button uses gradient AI button style
- [ ] Bulk processing shows per-file progress with AI styling
- [ ] Both modals match the glass morphism panel pattern from Roadside AI
- [ ] Dark theme consistent, no light-mode artifacts
- [ ] No regression in upload/extraction functionality

## Implementation Summary

### FN-358 (subtask): Redesign Auto-Create from PDF modal

**What was done:**
- Added AI hero pill ("AI-Powered" with `auto_awesome` icon) above the title
- Added `neurology` brain icon to the title
- Added subtitle explaining the AI extraction feature
- Redesigned close button with glass morphism hover effect
- Replaced plain dropzone with AI-branded drop area:
  - Large `cloud_upload` icon centerpiece
  - Cyan dashed border with hover glow effect
  - Green state when file is selected (with `description` icon + filename)
  - AI extraction state with pulsing `neurology` icon and indeterminate progress bar
- Replaced generic buttons with AI gradient buttons:
  - "Run Extraction" → indigo→cyan gradient with `auto_awesome` icon, hover lift + glow
  - "Close" → subtle glass secondary button
- Redesigned error/warning alerts with icon + pill styling

**Files Changed:**
- `frontend/src/app/components/loads-dashboard/loads-dashboard.component.html` — modal template redesign
- `frontend/src/app/components/loads-dashboard/loads-dashboard.component.scss` — new AI-themed styles (~250 lines)

**Key Decisions:**
- Used `.ai-` prefix for all new CSS classes to avoid conflicts with existing modal styles
- Reused existing modal-backdrop and modal container — only changed inner content
- File input uses absolute positioning over dropzone for full-area click target
- Processing animation uses CSS-only approach (no JS timers) for performance
