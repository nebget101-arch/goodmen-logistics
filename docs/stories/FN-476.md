# FN-476: AI-Powered Fuzzy Driver Name Matching for FMCSA Inspections

## Requirement
When the FMCSA matching engine fails to match an inspection driver via CDL number or exact name lookup, use Claude AI as a fallback to fuzzy-match the FMCSA driver name against fleet drivers. Handles abbreviations, nicknames, misspellings, and last-name-first formatting common in FMCSA reports.

## Acceptance Criteria
- [x] POST /api/ai/fmcsa/match-driver works with Claude API
- [x] Handles abbreviations, nicknames, misspellings, last-name-first format
- [x] Returns confidence 0-1 with reasoning
- [x] Auto-match >0.85, suggest 0.5-0.85, no match <0.5
- [x] Integrated as fallback in FMCSA matching flow (CDL -> name-exact -> AI fuzzy)
- [x] Results stored in match_details JSONB column on fmcsa_inspection_history

## Architecture

### Matching Chain (priority order)
1. **CDL number** (confidence 0.95) -- exact match via driver_licenses table
2. **Name exact** (confidence 0.85) -- case-insensitive first+last match
3. **Name reversed** (confidence 0.75) -- LAST, FIRST format detection
4. **AI fuzzy** (confidence varies) -- Claude-powered nickname/abbreviation/misspelling handling

### AI Endpoint
```
POST /api/ai/fmcsa/match-driver
```

**Request:**
```json
{
  "fmcsaDriverName": "HORTON, TYRON D",
  "fleetDrivers": [
    { "id": "uuid", "first_name": "Tyron", "last_name": "Horton", "cdl_number": "..." }
  ]
}
```

**Response:**
```json
{
  "success": true,
  "match": { "driverId": "uuid", "confidence": 0.95, "reasoning": "..." },
  "status": "auto_match",
  "candidates": [...]
}
```

### Confidence Thresholds
| Range | Status | Action |
|-------|--------|--------|
| >= 0.85 | auto_match | Automatically assign driver |
| 0.50 - 0.84 | suggest | Show suggestion, needs human review |
| < 0.50 | no_match | No viable candidate found |

### Safeguards
- 15-second timeout on AI service calls (matching continues without AI on timeout)
- AI errors are logged but never break the matching chain
- Fleet driver list capped at 200 to avoid oversized prompts
- match_details JSONB stores full AI response for audit trail
